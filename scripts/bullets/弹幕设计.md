## 关于弹幕的一切

远程单位会发出弹幕

当前有三种弹幕类型：
普通：直接造成伤害
燃烧：命中后持续造成伤害
减速：降低命中目标的移速和攻速
冰冻：使命中单位无法移动，但是不影响攻击

弹幕有归属属性，弹反后会直接射向所属
例如敌人发出弹幕，玩家弹反，弹反后直接射向该敌人，此时该弹幕所属于玩家
同理玩家的弹幕也会被敌人弹反

弹幕有速度和加速度，在发射后会加速到最大速度

弹反时保留速度

## 重构后架构

### 类职责总览

| 类 | 类型 | 职责 |
|----|------|------|
| `BulletData` | Resource | 纯数据：弹幕类型、伤害、速度曲线、状态参数 |
| `StatusEffect` | Resource | 纯数据：状态类型、持续、跳伤间隔、强度 |
| `Bullet` | Area2D | 弹幕基类：飞行、碰撞、弹反、加速度 |
| `BulletFactory` | Node | 对象池：预加载、生成、回收、清理 |
| `StatusComponent` | Node | 状态管理：施加、跳伤、减速/冰冻修饰 |
| `HitboxComponent` | Area2D | 攻击/弹反判定：命中伤害、弹反反转 |

### 完整数据流

```
BulletData + PackedScene
        │
        ▼
BulletFactory.spawn(data, scene, pos, dir, shooter)
        │
        ▼
Bullet.activate()  ──  _process (加速度到 max_speed)
        │
   ┌────┴────┐
   ▼         ▼
 撞墙      命中实体
   │         │
   │    ┌────┴──────────┐
   │    ▼               ▼
   │ HurtboxComponent  HitboxComponent (parry_owner != null)
   │    │               │
   │    ▼               ▼
   │ take_damage()    Bullet.parry(new_owner)
   │ + apply_effect()   │
   │    │               ▼
   │    ▼          方向反转, owner切换, 速度保留
   │ StatusComponent
   │    │
   │    ├── BURN: 每 tick_interval 造成 power 伤害
   │    ├── SLOW: speed_multiplier 降为 power
   │    └── FREEZE: can_move = false
   │
   ▼
deactivate() → 回池
```

### BulletData 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `bullet_type` | enum {NORMAL, BURN, SLOW, FREEZE} | 弹幕类型 |
| `damage` | int | 命中伤害 |
| `initial_speed` | float | 初始速度 |
| `max_speed` | float | 最大速度 |
| `acceleration` | float | 加速度（每秒） |
| `status_duration` | float | 状态持续时间 |
| `status_power` | float | 状态强度（燃烧=每跳伤害, 减速=速度倍率） |

### Bullet 关键 API

```gdscript
func activate(data: BulletData, pos: Vector2, dir: Vector2, shooter: Node2D)
func deactivate()
func parry(new_owner: Node2D)  # 反转方向、切换归属、保留速度
```

### BulletFactory 关键 API

```gdscript
func preload_pool(scene: PackedScene, amount: int = 20)
func spawn(data: BulletData, scene: PackedScene, pos, dir, shooter) -> Bullet
func spawn_by_path(data: BulletData, scene_path: String, pos, dir, shooter) -> Bullet
func return_bullet(bullet: Bullet)
func clear_all_bullets_of_type(scene: PackedScene)
```

### 扩展新弹幕类型

1. 在 `BulletData.BulletType` 枚举中添加新类型
2. 若需要新状态效果，在 `StatusEffect.Type` 枚举中添加
3. 在 `StatusComponent` 中实现对应的效果处理逻辑
4. 创建 `.tres` 资源文件配置参数，或运行时 `BulletData.new()` 设置字段
5. 调用 `BulletFactory.spawn()` 发射
